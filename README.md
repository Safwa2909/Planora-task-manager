# 🚀 Planora – Task Management Web Application

Planora is a modern task management web application built using **Flask** that helps users organize, track, and manage daily tasks efficiently through an interactive dashboard.

This project was developed as part of a **Full Stack Web Development Internship Task**.

---

## 🌟 Features

✅ User Authentication (Register & Login)  
✅ Secure User Sessions  
✅ Create Tasks with Description  
✅ Task Status Management (To-Do / Progress / Done)  
✅ Priority Levels (Low / Medium / High)  
✅ Due Date Tracking  
✅ Countdown Days Remaining  
✅ Delete Tasks  
✅ Responsive Modern UI  
✅ Dark Theme Dashboard
